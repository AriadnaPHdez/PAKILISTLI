Datos privados que recibimos y recolectamos

Para poder utilizar todas las funciones de PAKILISTLI en su versión móvil se requiere de un usuario registrado en el sistema, dichos datos son necesarios para comprobar que un usuario legítimo está haciendo uso de la aplicación. Sin embargo, para la versión web no se almacenan datos o Información de cookies.

Los siguientes son los datos que se recolectan al momento de registrarse como:

  • Nombre(s), apellido paterno, apellido materno.
  • Email que funciona como usuario para ingresar al sistema y visualizar de manera personalizada la sesión en la app.
  • Una contraseña única para completar las credenciales de acceso.
  • Número de teléfono (válido de 10 dígitos).
  • Pin de acceso de caso de perder la contraseña.
  • Fecha de nacimiento
  • Sexo (includo el no binario).

Cómo usamos los datos que recolectamos

Se requieren para realizar las siguientes validaciones, funciones o procesos dentro del sistema:

  1. Para brindar seguridad a los datos del usuario se necesitan credenciales de acceso para identificar al usuario cuando ingresa a la aplicación y mientras su sesión está activa.

  2. Para bloquear los servicios a un usuario no autorizado o con credenciales incorrectas.

  3. En caso de enviar un correo con comentarios y reportes se utilizará el correo de contacto para dar seguimiento a la solicitud.

  4. Para tener un panorama completo de las necedidades de los usuarios de la aplicación, como diagnóstico de problemas, enfermedades registradas, uso de los diferentes servicios y de esta manera tener la capacidad de mejorar el servicio para todos los usuarios.

  5. Para permitir que los usuarios verifiquen la Información que han introducido al sistema y tengan la posibilidad de realizar cambios de ser necesario.

Consentimiento para usar los datos ingresados en el sistema

Usando cualquiera de los servicios o ingresando Información personal por medio de PAKILISTLI, usted consiente el almacenamiento, transferencia a múltiples servidores o ubicaciones de almacenamiento de PAKILISTLI y el uso de su información definida en esta declaración de privacidad. Si usted no consiente el uso de su información personal, por favor absténgase de utilizar los servicios para garantizar que sus datos no serán usados de alguna manera.
